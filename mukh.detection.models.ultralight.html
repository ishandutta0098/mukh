Ultra-Light face detector implementation.

A 1MB size face detector optimized for edge devices. Provides bounding box detection without landmarks.

detect(image_path: str) → List[FaceDetection][source]

Detects faces in the given image using Ultra-Light model.

The image is resized to self.input_size for inference and results are scaled back to original image size.

Parameters:

image_path – Path to the input image.

Returns:

List of detected faces, each containing:

  • bbox: BoundingBox with coordinates and confidence

  • landmarks: None (Ultralight doesn’t provide landmarks)

Return type:

List[FaceDetection]
