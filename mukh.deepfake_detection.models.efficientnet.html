mukh.deepfake_detection.models.efficientnet.efficientnet_detector module

EfficientNet deepfake detection model implementation.

This module implements a deepfake detection model using EfficientNet architecture based on the fornet library implementation, with explainability features through GradCAM visualization.

GitHub: https://github.com/polimi-ispl/icpr2020dfdc
